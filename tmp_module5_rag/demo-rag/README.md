# demo-rag

RAG Agent template generated by `create-agent-app`.

## What you get

- ChromaDB vector store persisted locally
- Sentence-transformer embeddings (`all-MiniLM-L6-v2`)
- Ingestion pipeline (`rag/ingest.py`)
- Retriever tool integrated with a LangGraph agent
- FastAPI endpoint (`/invoke`)

## Quickstart

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
cp .env.example .env
# Configure provider key(s) in .env
python main.py
```
