# demo-single

Single Agent template generated by `create-agent-app`.

## What this project includes

- LangGraph single-agent workflow with tool-calling
- Calculator tool example
- FastAPI service with `/invoke` endpoint
- Provider-agnostic config for Groq, Gemini, Azure OpenAI, and Ollama

## Quickstart

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
cp .env.example .env
# Fill provider credentials in .env
python main.py
```

## Run API

```bash
uvicorn api.main:app --reload --port 8000
```

Then call:

```bash
curl -X POST http://localhost:8000/invoke -H "Content-Type: application/json" -d '{"query":"What is 23 * 9?"}'
```
