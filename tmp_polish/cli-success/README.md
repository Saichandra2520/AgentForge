# cli-success

Multi-Agent Supervisor template generated by `create-agent-app`.

## Template Overview

This scaffold uses a supervisor agent to route work between:
- `researcher` worker (collects notes)
- `writer` worker (produces the final response)

Best for tasks that benefit from staged reasoning.

## Architecture

```text
User Task
   |
   v
[Supervisor Node]
   | route=researcher
   v
[Researcher Worker] --> [Search Tool]
   |
   v
[Supervisor Node]
   | route=writer
   v
[Writer Worker]
   |
   v
[Supervisor Node] -> route=finish -> [END]
```

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
cp .env.example .env
# Fill provider credentials in .env
python main.py
```

## API Mode

```bash
uvicorn api.main:app --reload --port 8000
```
