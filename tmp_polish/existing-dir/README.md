# existing-dir

Single Agent template generated by `create-agent-app`.

## Template Overview

Build a single assistant that can call tools and respond through either CLI or FastAPI.
Best for focused workflows where one agent owns the full task.

## Architecture

```text
User Input
   |
   v
[LangGraph StateGraph]
   |
   v
[Agent Node] --tool call?--> [ToolNode: calculator] --back--> [Agent Node]
   |
   +-------------------------------> [END]
```

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
cp .env.example .env
# Fill provider credentials in .env
python main.py
```

## API Mode

```bash
uvicorn api.main:app --reload --port 8000
```

```bash
curl -X POST http://localhost:8000/invoke -H "Content-Type: application/json" -d '{"query":"What is 23 * 9?"}'
```
