# demo-multi

Generated by `create-agent-app` using the `multi_agent` template.

## Quickstart

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
cp .env.example .env
# Set provider credentials in .env
python main.py
```

## Optional API mode

```bash
uvicorn api.main:app --reload --port 8000
```
